Purpose

These drivers provide a set of functions to manage:

  • LEDs, push-button, and COM ports
  • OctoSPI NOR Flash memory
  • HexaSPI PSRAM memory
  • LCD
  • Touch Screen

Note that stm32u5g9j_discovery_conf_template.h file must be copied in user application as stm32u5g9j_discovery_conf.h with optional configuration update.
